# nodejs-crud-api

## To kill port

```bash     
lsof -i:3010
``` 
```bash 
kill -9 $PORT
```

## To start server  

```bash
npm run start:dev   
```    

## To start server in cluster mode

```bash
npm run start:multi   
```   

## To start server in production mode

```bash
npm run start:prod
```

## To build

```bash
npm run build
```

## To test  

```bash
npm run test
```
